# Mr.Gank
程序员的好朋友，上下班途中的技术阅读器，提供IT技术分享及妹子图分享给好友，可以方便收藏喜欢的文章

![](http://7xplt3.com1.z0.glb.clouddn.com/20180119151635014238448.png)

## 功能

1. 查看技术分享文章
2. 保存妹子图及分享给微信QQ好友
3. 搜索技术文章
4. 收藏好文
